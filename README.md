Esta aplicacion se utiliza para crear las tablas de multiplicar;      
```
--help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Esta es la base que debemos de mandar para que se haga la
                 multiplicacion.                             [number] [required]
  -l, --listar                                        [boolean] [default: false]
  -h, --hasta                                                           [number]
```
